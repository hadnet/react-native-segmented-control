# @hadnet/react-native-segmented-control

A fully customizable iOS like segmented control component for RN powered by Reanimated 2/3

## Installation

```sh
npm install @hadnet/react-native-segmented-control
```

## Usage

```js
import { ReactNativeSegmentedControlView } from "@hadnet/react-native-segmented-control";

// ...

<ReactNativeSegmentedControlView color="tomato" />
```
